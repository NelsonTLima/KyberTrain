# KyberTrain
KyberTrain é um protótipo feito em React de uma WebApp de cursos focada no aprendizado de CyberSegurança.

## Como Rodar?
### Instale as dependências
```
npm i
```
### Inicie o servidor Express
```
npm run express
```
### Inicie o servidor Vite
```
npm run build
```
